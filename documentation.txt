#1. classes.py

Базовый класс Base
Базовый класс для всех моделей данных, наследуемый от DeclarativeBase SQLAlchemy. 
Обеспечивает декларативный стиль определения моделей и управление метаданными.

Класс User - Пользователь, который зашел в бот
Поле    Тип Описание
id      int id пользователя в бд
tgid	str	id пользователя в тг

Класс Word - Слово (пара: русский текст-английский перевод)
Поле	        Тип	Описание
id	            int	id слова
rus_word	    str	Русское слово или фраза
eng_translate	str	Перевод на английский язык

Класс UsersWords - Связь пользователей и слов
Поле	Тип	Описание
id	    int	id связи
user_id	int	Внешний ключ, ссылающийся на users.id
word_id	int	Внешний ключ, ссылающийся на words.id

Связи между таблицами
Пользователь может иметь множество слов (через UsersWords)
Слово может принадлежать множеству пользователей (через UsersWords)
Тип связи: многие-ко-многим с использованием ассоциативной таблицы users_words

#2. inserting_base_to_db.py
Функции:
1) inserting_to_word_table - вставляет слово в таблицу words
2) inserting_to_user_table - вставляет id пользователя в таблицу users
3) inserting_to_userswords_table - вставляет запись в связующую таблицу users_words
4) get_user_id_by_tgid - получение id пользователя в системе по его id в тг
5)  get_word_id_by_rus_word - получение id слова по его русскому названию
